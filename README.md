# Amar Portfolio Software

get your portfolio in secs with best ui & admin access.




## Features

- Responsible ui with dark/white mode
- easy to customize in dashboard

### Pages:
 - home
 - socials
 - projects
 - contact
 - discover



## Deployment
You can easily host this in any node.js web hosting service.
